1.타임리프(Thymeleaf)

스피링부트에서 HTML랜더링 시 사용 하는 템플릿 엔진
컨트롤러에서 넘겨준 데이터를 HTML에 "바인딩"할 수 있는것이 특징.
JSP를 대체하는 현대적인 방법

2.타임리프 시작 필수셋팅

  1. 타임리프 디펜던시 : build.gradle or pom.xml
  2. 템플릿 파일 위치 설정 : resources/template 폴더에 .html 파일저장할것.
  3. HTML 파일에 타임리프 네임스페이스 선언 : HTML은 기본적으로 타임리프속성 몰라서
